Chapter 6 - Troubleshooting and Security
 

Troubleshooting

 
We will discuss some of the common issue faced by users while performing tasks on Batch Orchestrator:

Product Category Creation Fails

 

Users are unable to create product category for invalid MR:

  1. When MR is present in the server point but does not exist in Control Builder. This may occur due to error while deleting MR, of after migration or after a bulk delete
  2. No Data Block is associated to MR
  3. When MR is associated with a Data Block, but there is no formula parameter that exist and is configured with only report parameter
  4. When MR is in Red Status in Control Builder
 

Security

 
The Batch Orchestrator uses SSL Certificates to secure all the data while passing from the browser to the Experion Server. Users can verify this by checking the URL after Installation of the Batch Orchestrator. It must begin with "https", this indicates that the site is secured using SSL certificate (s stands for secure).
 
Please refer to the "Software Installation User Guide" and "Network Best Practices", for more information.
