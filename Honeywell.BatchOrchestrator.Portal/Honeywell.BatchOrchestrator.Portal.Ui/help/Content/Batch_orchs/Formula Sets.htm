Chapter 5 - Formula Sets
 
Previously in Experion, where you create multiple products with different variants that will produce in the market. Today to manufacture those variants, we end up developing multiple master recipes. Further, each controller and master recipe becomes a tailor for that product.
Otherwise, if you need to create different variants using a single master recipe, managing formulation for each control recipe created for different variant will become essential to ensure that right material, process, specification, control parameter is being used for the right product. Therefore, as a result we end up creating multiple master recipes for different product variants.
 
With Formula Sets, we can control the number of master recipes created by enabling you to define them into product categories. Hence, in formula set you will create a single master recipe and link or use them for different products manufactured under the same batch. Where the task of managing the formulation is included as a feature in formula sets.
