Change in Experion

 

When a user introduces some changes to a product categor's master recipe from data block in Control Builder, these changes will get refelected to the Batch Orgestrator and can we reviewed. For example, if there is a change in the default minimum and maximum values of some formula parameters of a master recipe, user will get the notification to review and validate them.

 

This allows user to further make necessary changes in the Product Formula as per their requirment:





Change are indicated through "Action Required" icon, which directs the user to Experion Modification Window where user can review them.









Click on next, and Experion Modification Window will display the list of all the products that are affected by these changes.





Here user can select the products these changes are applied and click on "Apply Changes". This will revert the products to "draft" status. User can further edit, review and validate these changes and final approve it.
