Chapter 2 - Introduction
 
Experion Batch is Honeywell’s new automation solution for batch applications. Extending Honeywell’s leading technology with innovative design, creates a distributed controller-based batch solution with a reduced footprint. Batch Solutions focuses on faster project execution, better operations, easier compliances and production flexibility. Experion Batch can be used in a variety of unit-based applications
Batch Orchestrator is a unique operation visualization, providing operator guidance to the users for Batch Processes for better operations. The intent of this program is to create better and simpler standard reports, integrate views of different operations, provide operational excellence with quick qualification and validation and to flexibly create new product variants. It will help users to maintain compliance, bring the products to market much faster and improve operational values of the products.
In this release we will discuss the Batch Orchestrator user layout, User profile management, what is formula and formula sets, creating formula sets and product categories and troubleshooting of some commen issues.
